    Commands¶

    • npm create vite@latest - Create a new Vite project.
    • npm install tailwindcss @tailwindcss/vite - Install Tailwind CSS.
    • npm install @clerk/clerk-react - Install @clerk/clerk-react.
    • npm install react-router @clerk/react-router - Install react-router and @clerk/react-router.
    • BUILD_SHA_HERE - test

    Installed Dependencies¶

    • TailwindCss v4
    • mkdocs serve - Start the live-reloading docs server.
    • mkdocs build - Build the documentation site.
    • mkdocs -h - Print help message and exit.
